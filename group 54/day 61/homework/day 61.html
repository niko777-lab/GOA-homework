2 სად და რატომ ვიყენებთ JavaScript-ს?

JavaScript გამოიყენება ვებ-გვერდებზე დინამიკური და ინტერაქტიული ფუნქციონალის დასამატებლად. ის საშუალებას გვაძლევს, ვუპასუხოთ მომხმარებლის ქმედებებს, შევცვალოთ გვერდის ელემენტები, შევასრულოთ გამოთვლები და ვიმუშაოთ სერვერთან ასინქრონულად.

3 რა არის MERN Stack და მისი ნაწილები

  • MongoDB - მონაცემთა ბაზა, სადაც ინახება აპლიკაციის მონაცემები.
  • Express.js - Node.js-ისთვის განკუთვნილი ვებ-ფრემვორკი, რომელიც ამუშავებს სერვერის ლოგიკას.
  • React.js - Frontend ბიბლიოთეკა, რომელიც ქმნის მომხმარებლის ინტერფეისს.
  • Node.js - სერვერის მხარის პლატფორმა, რომელიც საშუალებას აძლევს JS-ს იმუშაოს სერვერზე.

4 JS-ის კოდის დაწერის სხვადასხვა გზა

  1. Inline JavaScript (HTML ატრიბუტში, მაგ: onclick)
  2. Internal JavaScript (HTML-ის <script> თეგში)
  3. External JavaScript (გარე .js ფაილიდან)
